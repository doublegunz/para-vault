# Learn SQL: Query, Manage, and Design Databases from Scratch

## Deskripsi
Learn SQL from zero using MySQL. Master SELECT, WHERE, JOIN, GROUP BY, subqueries, and database design. Build a complete bookstore database step by step.

## Konten
Every application stores data in a database, and SQL (Structured Query Language) is how you talk to it. Whether you are building a web application, analyzing data, or managing a business system, SQL is the universal language for creating, reading, updating, and deleting data in relational databases.

This course teaches SQL from absolute zero using MySQL as the database engine. You will start with simple SELECT queries, then progressively learn filtering, sorting, aggregation, data manipulation, table design, relationships, joins, subqueries, and views. Every lesson builds on a single project: a **bookstore database** with books, authors, customers, and orders.

Unlike courses that teach SQL in isolation with random examples, every query in this course has a practical purpose. You are not just learning syntax; you are building and querying a real database that grows more complex with each lesson.

**This course is for you if:**
- You have never written a SQL query before
- You are a web developer who wants to understand what happens inside the database
- You are a student taking a database course at university
- You want to learn data querying for analysis or reporting
- You need SQL skills before learning an ORM like Eloquent, Hibernate, or GORM

**By the end, you will have:**
- The ability to write SELECT queries with filtering, sorting, and aggregation
- Experience with INSERT, UPDATE, and DELETE for data manipulation
- Knowledge of table design: data types, constraints, primary keys, and foreign keys
- Working experience with JOINs for combining data across related tables
- Understanding of subqueries and views for complex queries
- A complete bookstore database that demonstrates real-world SQL patterns
