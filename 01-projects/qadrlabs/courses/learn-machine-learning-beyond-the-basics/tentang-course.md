# Machine Learning: Beyond the Basics

## Deskripsi
Level up your machine learning skills in Google Colab. Build leak-free pipelines, engineer features, use random forests and boosting, tune with cross-validation, handle imbalanced data, and apply clustering and PCA with scikit-learn.

## Konten
You finished the beginner course. You can build regression and classification models, evaluate them, and prepare messy data. Now it is time to go further and learn the techniques that turn a working model into a strong, reliable one.

This course picks up right where "Learn Machine Learning for Beginners" left off. You still work entirely in Google Colab with scikit-learn, and you still focus on classical machine learning, not deep learning. The difference is depth. You will bundle preprocessing and modeling into pipelines that never leak data, engineer better features, and reach for more powerful models like random forests, gradient boosting, and support vector machines. You will learn to evaluate and tune models properly with cross-validation and grid search, handle imbalanced data, and read advanced metrics like ROC and AUC. Finally you will step into unsupervised learning with clustering and dimensionality reduction, and learn to save a finished model for reuse.

Each module builds on the last, and the course ends with a capstone where you assemble a complete, tuned pipeline from start to finish. By the end you will be comfortable with the intermediate toolkit that real machine learning projects rely on.

**This course is for you if:**
- You completed the beginner course or know the basics (the ML workflow, regression, classification, evaluation, and data prep)
- You can train a scikit-learn model and measure it, but want to make it genuinely good
- You keep hearing terms like pipeline, cross-validation, random forest, and grid search, and want to actually use them
- You are ready for intermediate techniques but not yet for deep learning

**Prerequisites:**
- Completed "[Learn Machine Learning for Beginners](https://qadrlabs.com/course/learn-machine-learning-for-beginners)" or equivalent knowledge
- Comfortable with: the train/test workflow, `fit`/`predict`, basic metrics, handling missing values, encoding, and scaling
- A Google account to use Google Colab (free)

**By the end, you will have:**
- The ability to build robust scikit-learn pipelines with `ColumnTransformer` that prevent data leakage
- Practical feature engineering skills
- Hands-on experience with random forests, gradient boosting, and support vector machines
- A solid grasp of cross-validation and hyperparameter tuning with grid and randomized search
- The skills to evaluate with ROC and AUC and to handle imbalanced datasets
- An introduction to unsupervised learning: K-means clustering and PCA
- The ability to save, load, and reuse a trained model
- A complete, tuned, end-to-end machine learning project
