# Learn PHP for Beginners: From Zero to Database-Driven Web Pages

## Deskripsi
Learn PHP from scratch. Master variables, arrays, functions, forms, and database interaction with PDO. Build your first dynamic web pages step by step.

## Konten
Every PHP framework, every CMS, every dynamic website you have ever used is built on PHP fundamentals. Before you can build applications with Laravel, Symfony, or CodeIgniter, you need to understand the language they are written in.

This course starts from absolute zero. You do not need any prior programming experience. By the end, you will be writing PHP that connects to a database, handles form submissions, and generates dynamic HTML pages. These are the exact skills you need before stepping into object-oriented programming and frameworks.

Instead of teaching concepts in a vacuum, we build toward a real goal throughout the course: a simple journal page where you can create, read, update, and delete entries. The project starts small (displaying variables on a page) and grows lesson by lesson until you have a fully working database-driven web page.

**This course is for you if:**
- You have never written a line of PHP before
- You know some HTML and want to make your pages dynamic
- You tried learning a PHP framework but felt lost because the language fundamentals were missing
- You want a solid foundation before learning OOP and frameworks

**By the end, you will have:**
- A clear understanding of PHP syntax, data types, control structures, loops, arrays, and functions
- The ability to process HTML form submissions with validation
- Hands-on experience with MySQL using PDO and prepared statements
- A working journal page that reads and writes data to a database
- The confidence to move on to PHP OOP and eventually a framework
