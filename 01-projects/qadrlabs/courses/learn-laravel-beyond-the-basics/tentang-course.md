# Learn Laravel: Beyond the Basics

## Deskripsi
Level up your Laravel skills. Extend Catatku with relationships, authorization, file uploads, REST APIs, Pest testing, queues, events, and deployment.

## Konten
You have built Catatku, a personal journal application, in the beginner course. You learned routes, controllers, the Entry model, Blade components, CRUD operations, and basic authentication. Now it is time to go deeper.

This course extends Catatku with features that every production Laravel application needs. You will add comments to entries (one-to-many relationships), tags (many-to-many), authorization so users can only modify their own entries, file uploads for cover images, email notifications, a REST API with Sanctum authentication, automated testing with Pest, background job processing with queues, and deployment to production.

Each lesson builds on the previous one, adding one feature or concept at a time. The Entry model, x-layout component, and authentication system from the beginner course are the foundation for everything in this course.

**Prerequisites:**
- Completed the "[Learn Laravel for Beginners](https://qadrlabs.com/course/learn-laravel-for-beginners)" course (Catatku with Entry model)
- Comfortable with: routing, controllers, Eloquent basics, Blade templates (x-layout, x-entry-card), CRUD, basic auth
- Laravel 13 development environment

**By the end, you will have:**
- Eloquent relationships: one-to-many (Entry has Comments), many-to-many (Entry has Tags)
- Query scopes, accessors, mutators, eager loading, soft deletes, pagination
- Authorization with Gates and Policies
- File upload with Storage facade
- Email notifications with Mailables
- A REST API with Sanctum token authentication
- Automated tests with Pest
- Background jobs with queues, events and listeners
- Blade components and Tailwind CSS integration
- Production deployment knowledge
