# Learn CodeIgniter 4 for Beginners: Build a Personal Journal App from Scratch

## Deskripsi
Learn CodeIgniter 4 by building Catatku, a personal journal app with full CRUD, user authentication, and data protection. From zero to working application.

## Konten
Most CodeIgniter tutorials teach concepts in isolation. You learn routing in one lesson, models in another, and authentication somewhere else, but you never see how they all connect inside a real application. This course takes a different approach.

You will build **Catatku**, a personal journal application, from an empty project to a fully working app across 12 structured lessons. Every concept is introduced at the moment you need it, applied immediately, and built upon in the next lesson. By the end, you will not just know CodeIgniter syntax. You will understand why the code is organized the way it is.

The course follows CodeIgniter 4 with PHP 8.3 and covers the framework's modern conventions, including models with `$allowedFields`, Filters for route protection, and the built-in view layout system. No starter kits or scaffolding. You write every line yourself, so nothing feels like magic.

**This course is for you if:**
- You understand basic PHP (variables, functions, arrays, conditionals) but have never used a framework
- You have tried CodeIgniter before but got confused about how all the pieces fit together
- You want a portfolio-ready project, not just a collection of disconnected exercises

**By the end, you will have:**
- A working application you can demo, deploy, or extend on your own
- A solid mental model of how CodeIgniter processes a request from URL to database and back
- The confidence to read the official CodeIgniter documentation and understand it

No prior CodeIgniter experience required. Lesson 2 walks you through the full environment setup on Windows.
