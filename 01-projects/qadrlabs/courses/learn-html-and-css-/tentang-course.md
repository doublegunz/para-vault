# Learn HTML and CSS: Build Your First Website from Scratch

## Deskripsi
Learn HTML and CSS from zero. Master page structure, text, images, links, forms, the box model, Flexbox, Grid, and responsive design step by step.

## Konten
Every website on the internet is built with HTML and CSS. HTML (HyperText Markup Language) defines the structure and content of a page: headings, paragraphs, images, links, and forms. CSS (Cascading Style Sheets) controls how it looks: colors, fonts, spacing, layout, and responsive behavior.

This course teaches you both from absolute zero. No prior programming experience is required. You will start by creating a simple page with text, then progressively learn how to add images, links, tables, forms, colors, layouts, and responsive design. By the end, you will build a complete landing page that looks professional on both desktop and mobile.

Unlike other courses that teach HTML and CSS in isolation, every lesson in this course builds toward a practical goal: a multi-section landing page. This gives every tag and property a reason to exist.

**This course is for you if:**
- You have never written a line of HTML before
- You want to build websites but do not know where to start
- You need HTML/CSS fundamentals before learning JavaScript, PHP, or a framework
- You are a student taking a web development course

**By the end, you will have:**
- A solid understanding of HTML elements, attributes, and semantic structure
- The ability to style pages with CSS: colors, fonts, spacing, and backgrounds
- Experience with modern CSS layout: Flexbox and Grid
- A responsive landing page that works on desktop and mobile
- The foundation to learn JavaScript, Bootstrap, Tailwind CSS, or any web framework
