# Learn PHP OOP for Beginners: Build a Personal Journal App Without a Framework

## Deskripsi
Learn PHP OOP by building Catatku from scratch without any framework. Master classes, PDO, the Repository pattern, and MVC before moving to Laravel or Symfony.

## Konten
Frameworks are powerful, but jumping straight into one without understanding the fundamentals is like learning to drive in a race car. You can follow the instructions, but you do not really know what is happening under the hood.

This course takes you on the journey that frameworks skip. You will build **Catatku**, a personal journal application, using nothing but PHP and object-oriented programming. No Laravel, no Symfony, no CodeIgniter. Just PHP classes, PDO for database access, and patterns you design yourself.

By the end, you will not just know OOP syntax. You will understand *why* frameworks exist, because you will have built the same patterns they use: a front controller for routing, a Repository for database queries, a template system for views, and session-based authentication. When you move to a framework afterward, nothing will feel like magic. It will all feel familiar.

**This course is for you if:**
- You understand basic PHP (variables, functions, arrays, conditionals) but have never written a class
- You have tried a framework and felt lost because you did not understand the underlying concepts
- You want to build a solid OOP foundation before committing to a specific framework

**By the end, you will have:**
- A working application built entirely with PHP OOP and no framework
- A deep understanding of classes, objects, encapsulation, inheritance, interfaces, and dependency injection
- Firsthand experience with patterns that every PHP framework uses: Repository, Front Controller, and MVC
- The confidence to pick up Laravel, Symfony, or CodeIgniter and understand what they do under the hood

No prior OOP experience required. Lesson 2 walks you through the full environment setup.
