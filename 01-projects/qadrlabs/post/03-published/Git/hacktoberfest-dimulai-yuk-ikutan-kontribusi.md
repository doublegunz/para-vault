Semenjak masuk bulan Oktober, linimasa media sosial saya dipenuhi dengan postingan tentang Hacktoberfest. Bukan hanya di linimasa, di forum-forum pemrograman yang saya ikuti pun banyak yang membahas dan mengajak programmer lain untuk mengikuti **Hacktoberfest**. Dan akhirnya saya pun penasaran, apa sih **Hacktoberfest** itu?

## Apa itu Hacktoberfest?{#apa-itu-hacktoberfest}
Berdasarkan situs resminya, **Hacktoberfest** adalah Event yang diadakan DigitalOcean yang bekerja sama dengan beberapa partner, dalam rangka mendukung untuk berkontribusi di projek open-source. Event ini diadakan selama sebulan penuh lho! Dari tanggal 1 Oktober sampai tanggal 31 Oktober. Hacktoberfest ini pertama kali diadakan pada tahun 2014 dan tahun ini merupakan tahun ke-sepuluh (keterangan: update tulisan ini ditulis tahun 2023). Event ini diadakan dengan tujuan yang amat sangat sederhana, yaitu ikut berkontribusi ke komunitas open source yang selama ini telah meluangkan waktu mereka untuk memberi banyak manfaat.

Hacktoberfest ini terbuka untuk siapa saja, baik para pegiat komunitas perangkat lunak open source yang sudah lama aktif maupun untuk siapapun yang tertarik untuk berkontribusi dalam komunitas open source untuk pertama kalinya. Kalau kamu bertanya, misalkan belum bisa coding apa bisa ikutan Hacktoberfest? bisa kak, silakan ikutan. Karena berkontribusi di open source itu tidak selalu berhubungan dengan code. Hal sederhana seperti memperbaiki penulisan dokumentasi pun dapat bermanfaat untuk komunitas bukan?

## Apa itu Open Source?{#apa-itu-open-source}
Sebelumnya disebutkan hacktoberfest itu event untuk berkontribusi di projek open source. Mungkin ada yang bertanya apa itu open source? Open source di sini merujuk ke sumber kode yang dapat diakses oleh publik dan dapat dilihat oleh siapapun, dimodifikasi atau boleh juga untuk dipelajari. Project open source mendukung kolaborasi dan kebebasan untuk menggunakan software untuk tujuan apa pun.

## Apa manfaat mengikuti Hacktoberfest{#apa-manfaat-mengikuti-hacktoberfest}
Kalau tidak ada manfaat tentu tidak akan ada motivasi untuk mengikuti event ini. Ada banyak hal baru yang saya dapatkan ketika mengikuti event ini. Selain tahu cara untuk ikut berkontribusi, kita bisa mempelajari secara langsung cara menggunakan git, tahu seperti apa workflow yang sering digunakan oleh developer profesional, bagaimana cara untuk menangani project dan ada banyak hal menarik lainnya. Dan jangan lupa, kita juga dapat baju baru dan stiker limited-edition yang bisa kita tempel-tempel di laptop.

## Bagaimana cara mengikuti Hacktoberfest?{#bagaimana-cara-mengikuti-hacktoberfest}
Cara mengikuti Hacktoberfest ini pun cukup mudah. Kita diperbolehkan membuat Pull Request (PR) di setiap repositori atau project yang di-hosting di GitHub ataupun GitLab. 

Pull Request adalah proposal yang berisi perubahan kode yang kita ajukan di repositori tempat kita ikut berkontribusi. Selama projek itu bersifat public dan dihosting di GitHub atau GitLab, PR kita dianggap sebagai partisipasi mengikuti Hacktoberfest. Perlu diingat Hacktoberfest ini sangat mengedepankan kualitas, jadi pastikan kontribusi kita bermanfaat ya. Oh iya, Hacktoberfest ini diadakan dari tanggal 1 Oktober sampai tanggal 31 Oktober... meski agak telat, kita masih bisa ikut berpartisipasi di event ini.

Sebelum mendaftar, ada beberapa point yang mesti kita perhatikan dalam mengikuti event Hacktoberfest:

1. Everyone is welcome - Peserta dalam Hacktoberfest mewakili 119 negara dengan beragam keahlian. Event ini menyambut baik siapa saja yang ingin berkontribusi, baik para pegiat komunitas yang sudah aktif di komunitas open source, maupun orang-orang yang baru pertama kali ingin ikut berkontribusi.

2. Quantity is fun, Quality is key - Berpartisipasi dalam Hacktoberfest ini sangat bermanfaat untuk pengembangan diri, peluang untuk menjadi profesional dan membangun komunitas. Hal ini semua dimulai dengan memberikan kontribusi yang bermanfaat ke perangkat lunak open source.

3. Short term action, long term impact - Dengan Berpartisipasi dalam Hacktoberfest, mungkin kontribusi yang kita lakukan itu sederhana, tapi boleh jadi itu akan memiliki efek jangka panjang untuk komunitas dan teknologi open source, bahkan setelah event ini selesai.

## Langkah-langkah mendaftar Hacktoberfest.{#langkah-langkah-mendaftar-hacktoberfest}
Lalu, apa saja langkah-langkah mendaftar Hacktoberfest? *Check this out, ya!* 

1. Pastikan kita sudah punya akun GitHub atau GitLab.
2. Buka URL [https://hacktoberfest.com/auth/](https://hacktoberfest.com/auth/).

![gambar 1 - halaman inisiasi daftar hacktoberfest](https://cdn.jsdelivr.net/gh/gungunpriatna/tes-repositori@master/git/hacktoberfest-2025/1-pendaftaran-authorize-with-github.webp)

Tekan tombol **Initiate** di salah satu opsi, misalkan opsi **Authorize with gitbhub**. Setelah itu tunggu sampai proses inisiasi pendaftaran selesai. 
![gambar 2 - auth in progress](https://cdn.jsdelivr.net/gh/gungunpriatna/tes-repositori@master/git/hacktoberfest-2025/2-pendaftaran-auth-in-progress.webp).

3. Apabila telah selesai, halaman akan dialihkan ke halaman Pendaftaran Hacktoberfest

![gambar 3 - halaman pendaftaran](https://cdn.jsdelivr.net/gh/gungunpriatna/tes-repositori@master/git/hacktoberfest-2025/3-halaman-pendaftaran.webp)

Pada halaman ini lengkapi isian yang disediakan untuk berpartisipasi dalam event hacktoberfest. Pilih experience level, cara kontribusi kita (code atau non code), apakah terdaftar sebagai siswa atau mahasiswa, job role, nama negara dan opsi untuk dikirim email marketing. Apabila isian form sudah kita isi, jangan lupa checklist accept rule & terms untuk mengikuti hacktoberfest.

![gambar 4 accept rules and terms](https://cdn.jsdelivr.net/gh/gungunpriatna/tes-repositori@master/git/hacktoberfest-2025/3-pendaftaran-isi-profile-and-register.webp)

Setelah selesai, klik tombol **REGISTER** untuk melakukan proses pendaftaran.

Tunggu sampai proses pendaftaran selesai.

Setelah proses pendaftaran selesai, web akan dialihkan ke [halaman profile peserta](https://hacktoberfest.com/profile/) event hactoberfest.

![gambar 4 halaman profile](https://cdn.jsdelivr.net/gh/gungunpriatna/tes-repositori@master/git/hacktoberfest-2025/4-profile-page.webp)

Banyak sekali komunitas pengembang perangkat lunak open source yang sudah meluangkan waktu dan berdedikasi untuk memberi banyak manfaat untuk sekitarnya. Di event Hacktoberfest ini, mari sama-sama mendukung komunitas open source dengan cara ikut berkontribusi di projek open source. Dengan Berpartisipasi dalam Hacktoberfest, mungkin kontribusi yang kita lakukan itu sederhana, tapi boleh jadi itu akan memiliki efek jangka panjang untuk komunitas dan project open source, bahkan setelah event ini selesai.

Semangat berkarya, semangat berkontribusi! ^^
