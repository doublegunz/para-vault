Sebagian besar tema perkuliahan di semester ini adalah tentang analisis dan perancangan. Hampir semua mata kuliah membahas bagaimana cara merancang sistem yang akan dikembangkan, mendesain User Interface dari sistem yang akan dibuat nantinya dan juga pengetahuan yang diperlukan dalam perancangan. Karena pembahasannya tentang perancangan sistem, sudah pasti bahan yang diujikan tentang perancangan sistem juga. Oleh karena itu, di ujian akhir semester kemaren banyak tugas mengenai perancangan sistem, terutama tentang perancangan menggunakan UML.

Seperti yang kita ketahui, UML atau Unified Modeling Language adalah himpunan terstruktur dan teknik pemodelan desain program berorientasi objek (OOP) serta aplikasinya. Dan biasanya digunakan untuk mendokumentasikan, menspesifikasin dan membangun sistem perangkat lunak. Terdapat beberapa diagram yang biasa digunakan saat kita ingin memodelkan aplikasi diantaranya `Use Case Diagram`, `Activity Diagram`, `Sequence Diagram` ,`Class Diagram` dan aneka diagram lainnya.

Kali pertama membuat UML ini, saya membuatnya secara manual (corat-coret di buku). Karena hasilnya tidak terlalu rapi, saya berfikir bagaimana cara membuat UML supaya rapi.. apa ada program yang bisa dipakai? Setelah googling ternyata ada aplikasi yang sering digunakan dalam membuat UML, yaitu Microsoft Visio. Akan tetapi, Visio ini termasuk aplikasi yang berbayar, jadi kita mesti membeli lisensinya terlebih dahulu. Akhirnya, dosen saya pun menyarankan untuk membuatnya di website [draw.io](https://www.draw.io/index.html?offline=1). Apa sih draw.io itu? Ternyata draw.io itu software diagram online yang bisa kita gunakan secara gratis tanpa perlu mendaftar dan membayar. Di draw.io ini, kita bisa membuat flowcharts, process diagrams, org charts, UML, ER dan network diagrams.

Karena mesti online, saya berfikir apa draw.io ini bisa dipakai secara offline? Ya, kendala yang dihadapi mahasiswa saat itu adalah kesulitan untuk terkoneksi internet. Jadi saya coba eksplore draw.io dan ternyata dapat digunakan secara online. Bagaimana caranya? *Check this tips out ya!*

Ada dua cara supaya kita bisa menggunakan draw.io secara offline:

## Cara 1: Menggunakan Aplikasi Web dalam Mode Offline{#cara-1}
Untuk menggunakan aplikasi web draw.io secara offline itu cukup mudah. Kamu cukup buka browser kesayanganmu, lalu coba akses URL berikut ini: [https://app.diagrams.net/index.html?offline=1](https://app.diagrams.net/index.html?offline=1). Saat kamu mengakses URL tersebut, aplikasi draw.io bakalan diunduh di background. Nah, kamu bisa langsung pakai setelah file yang dibutuhkan sudah didownload (di background). Cukup sekali akses URL itu dan kamu sudah bisa pakai aplikasi draw.io secara offline.

![draw.io offline](https://1.bp.blogspot.com/-e3qMWmGNhgI/V5xGWlGt8kI/AAAAAAAAAbc/J4N2aJVugeExIY99BUkwpr4VatvJUc2_QCEw/w640-h306/cara%2B1.png)


## Cara 2: Menggunakan Aplikasi Chrome{#cara-2}
Untuk menggunakan cara yang kedua ini, sudah pasti kita mesti install Google Chrome terlebih dahulu di OS kita. Setelah Chrome sudah terinstall, kita bisa mengunjungi webstorenya chrome terus search 'draw.io desktop' atau bisa juga klik langsung di [sini](https://chrome.google.com/webstore/detail/drawio-desktop/pebppomjfocnoigkeepgbmcifnnlndla).

![ekstensi draw io](https://1.bp.blogspot.com/-9JZ952wgeuA/V5xGv2ze59I/AAAAAAAAAbg/wC4kHF89_2oWfc-jS4Wwo_GHbLteDH_GgCEw/w640-h382/gambar%2B1.png)

Klik tombol 'add to chrome' untuk menginstall draw.io Desktop. Setelah itu kamu bakalan punya icon Chrome App Launcher di OS kamu.

![add to chrome](https://2.bp.blogspot.com/-2fW05BM7q6U/V5xHhG1rgVI/AAAAAAAAAbo/q0PDXGOvamQZSravME0Ys3LIdLBpwwplgCLcB/w640-h301/gambar%2B2.png)

Klik icon nya nanti kamu bisa melihat aplikasi chrome yang sudah terinstall. Pilih draw.io Desktop untuk membukanya. Voila~ kamu sudah bisa pakai draw.io secara offline! ^^

![draw.io app](https://1.bp.blogspot.com/-ilRUGWkSmm4/V5xHwKmxPBI/AAAAAAAAAbw/JFkzjaUiaXwEdageeAafR8Y9Ozm_IJRMQCLcB/w374-h400/gambar%2B3.png)
