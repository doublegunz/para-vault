# Changelog

This page records the version history of the book. Software changes over time, and a book about software has to change with it. When a new edition fixes errors or updates code for a new framework version, the change is logged here so you always know which version you are reading and what changed.

If you bought the book through qadrlabs.com, Gumroad, or LemonSqueezy, you can re-download the latest version from the same place at no extra cost.

## Version History {#version-history}

**1.0.0 - 2026 - First Edition**

- Initial release.
- Based on Laravel 13 and PHP 8.3.
- Twelve chapters covering the full Catatku build: setup, routing, MVC, migrations, Eloquent, full CRUD, and authentication.
- Front matter: preface, how-to-use guide, and licensing.
- Back matter: Artisan cheat sheet, troubleshooting guide, glossary, source-code guide, full exercise solutions, and a guide to the follow-up course.

## How Updates Work {#how-updates-work}

When Laravel releases a new major version, conventions can shift. If a change affects the code in this book, a new edition is published and noted above. Bug fixes and clarifications from reader feedback are rolled into point releases. Either way, your purchase entitles you to the updated file from your original place of purchase.

If you spot something that looks out of date or incorrect, please report it through qadrlabs.com. Reader reports are the main reason each edition is more accurate than the last.
