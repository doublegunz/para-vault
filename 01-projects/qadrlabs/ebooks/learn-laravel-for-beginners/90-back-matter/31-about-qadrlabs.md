# About qadrlabs

**qadrlabs** publishes practical, build-something-real programming tutorials, courses, and books. The idea behind everything here is simple: you learn to build software by building software, not by collecting disconnected facts. So every piece of material centers on a real project, introduces each concept exactly when it is needed, and always explains the reasoning behind the code rather than just the syntax.

This book is one expression of that idea. Catatku is small on purpose, because a focused project lets you see how every part connects without drowning in scope. The same philosophy runs through the rest of what qadrlabs makes.

## More From qadrlabs {#more-from-qadrlabs}

If this book worked for you, here are good next steps to keep the momentum going. All of these live at **qadrlabs.com**.

**Learn Laravel: Beyond the Basics.** The direct sequel to this book. It extends the same Catatku app with database relationships, authorization through Gates and Policies, file uploads, email, a REST API with Sanctum, automated testing with Pest, queues and events, a real Vite plus Tailwind build, and deployment to production. If you finished this book and want to keep going, this is the next step. There is a full guide to it in the "Continue Learning" section of this book.

**The Learn Laravel for Beginners course.** This book grew out of the qadrlabs course of the same name. If you prefer learning by watching and following along lesson by lesson, the course covers the same Catatku project in that format.

**Follow-on tutorials and articles.** qadrlabs regularly publishes shorter, focused tutorials that go deeper into individual topics. These are a natural way to pick up the topics listed in the "What's Next" chapter one at a time.

**Whatever comes next.** New courses and books are always in progress. The best way to hear about them first is to follow qadrlabs at qadrlabs.com.
