# Ebook Production Meta

This file tracks the production status and structure of the ebook edition of the **Learn Laravel for Beginners** course. It is an internal working file and is not part of the ebook itself.

## Product Summary {#product-summary}

The ebook repackages the 12-lesson Catatku course into a standalone book that can be sold on qadrlabs.com and on Gumroad / LemonSqueezy. The core teaching content is adapted from the existing course lessons; the front matter, back matter, exercises, and reference appendices are written specifically for the book format.

## Structure {#structure}

The book is organized as front matter, six parts (twelve chapters), and back matter.

### Front Matter

- Cover
- Title Page
- Copyright and License
- About the Author (short)
- Preface
- How to Use This Book
- Table of Contents

### Core Chapters (6 parts, 12 chapters)

- Part I - Getting to Know Catatku (Ch 1-2)
- Part II - Laravel Foundations (Ch 3-4)
- Part III - Database and Model (Ch 5-6)
- Part IV - Core Features of Catatku (Ch 7-9)
- Part V - User Authentication (Ch 10-11)
- Part VI - What's Next (Ch 12)

Each core chapter (1-11) adds three ebook elements not present in the course: **Key Takeaways**, **Checkpoint**, and **Exercises**. Chapter 12 is a reflection chapter and adds Key Takeaways only.

### Back Matter

- Continue Learning: Learn Laravel Beyond the Basics (upsell to the sequel course)
- Appendix A - Artisan Command Cheat Sheet
- Appendix B - Troubleshooting Common Errors
- Appendix C - Glossary
- Appendix D - Getting the Source Code
- Exercise Solutions
- Thank You and Leave a Review
- About qadrlabs / Other Products
- Changelog
